# agy-gate

Cross-model code review for Claude Code, powered by the Antigravity CLI (`agy`) on a Gemini subscription. Sibling of [`codex-gate`](https://github.com/vitorlm/codex-plugin-cc) with an identical verdict contract.

> **Status:** 0.1.0 — under construction. macOS supported; Linux experimental (bwrap path unverified — see `docs/port-plan.md` gap #1).

Full docs land at v0.1.0 release (see `docs/tech-spec.md`).
